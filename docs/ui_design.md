# 安卓无障碍点击助手 - UI/UX 设计方案

**版本:** v1.0
**日期:** 2023年12月12日
**主题:** Ghost Macro (幽灵宏) - 现代、半透明、游戏极客风

---

## 1. 设计理念 (Design Philosophy)
*   **Ghost Macro (幽灵宏)**: 强调工具的“隐形”与“陪伴”。在不使用时，它应像幽灵一样透明、不可见，不遮挡游戏视线；在使用时，它应像实体按键一样响应迅速、位置精确。
*   **Glassmorphism (毛玻璃)**: 采用背景模糊的半透明材质，使悬浮窗能自然融合于各种游戏画面之上，既保持可读性又不破坏沉浸感。
*   **Dark Mode Priority (深色优先)**: 默认采用深色主题，符合游戏玩家习惯，减少夜间使用的视觉疲劳。

## 2. 界面结构 (Interface Structure)

### 2.1 软件主界面 (App Dashboard)
用户管理脚本和全局设置的入口。

*   **布局架构**: 单Activity + Jetpack Compose 导航。
*   **顶部栏 (Top Bar)**:
    *   **左侧**: App Logo & 标题 "AutoAction"。
    *   **右侧**: 
        *   **服务状态指示灯**: 🟢 (运行中) / 🔴 (停止)。点击可跳转无障碍设置。
        *   **全局设置**: ⚙️ 图标，进入偏好设置 (主题色、触觉反馈等)。
*   **核心区域 (Content - Tab: 脚本库)**:
    *   **卡片式列表**: 
        *   每个脚本展示为一个 `Surface` 卡片。
        *   **左侧**: 图标预览 (圆角矩形，显示自定义图片或首字母)。
        *   **中间**: 脚本名称 (粗体) + 简要信息 (如 "5个动作 | 循环无限")。
        *   **右侧**: **启用开关 (Switch)**。打开即在屏幕生成对应的“悬浮快捷键”。
    *   **操作**: 
        *   点击卡片 -> 进入 **编辑器**。
        *   长按卡片 -> 弹出菜单 (重命名、复制、删除)。
    *   **FAB (悬浮动作按钮)**: 
        *   位于右下角，大号 Gradient 渐变色按钮。
        *   点击弹出: "➕ 新建脚本", "🔴 开始录制"。

### 2.2 脚本编辑器 (Script Editor)
专注于动作编排的可视化界面。

*   **垂直时间轴 (Vertical Timeline)**:
    *   摒弃枯燥的列表，使用连接线串联的节点图。
    *   **动作节点**: 圆形图标 (点击/滑动/长按)，旁注坐标 (x,y)。
    *   **延时连线**: 节点之间的连线长度代表延时短，线上标注时间 (e.g., "Wait 200ms")。
*   **交互逻辑**:
    *   **点击节点**: 底部弹出 **参数配置面板 (BottomSheet)**。
        *   可修改: 坐标(支持微调)、时长、**随机偏差范围**。
    *   **拖拽排序**: 长按节点可上下拖动改变执行顺序。
    *   **左滑删除**: 快速移除节点。
*   **外观配置 (Shortcut Preview)**:
    *   编辑器顶部提供“悬浮键预览”。
    *   设置项: 图标更换、透明度滑块 (0-100%)、尺寸缩放。

### 2.3 悬浮窗系统 (Floating System)
用户在游戏场景下的主要交互界面。

#### A. 全局控制条 (Global Control Bar) - "隐形指挥官"
*   **常态**: 屏幕左侧边缘的一个半透明小胶囊 (Pill)，仅显示 Logo 或折叠箭头。
*   **展开态**: 点击胶囊，横向展开毛玻璃面板。
    *   🔴 **录制**: 进入全屏录制模式。
    *   👁️ **显/隐**: 一键隐藏/显示所有脚本快捷键 (游戏高强度作战时防误触)。
    *   📂 **列表**: 弹窗快速启用/禁用脚本。
    *   ❌ **退出**: 关闭服务。

#### B. 独立脚本快捷键 (Script Shortcuts) - "技能贴纸"
*   **形态**: 无边框图标，自由漂浮。
*   **视觉反馈**:
    *   **默认**: 半透明 (用户设定，如 40% Alpha)。
    *   **按下**: 不透明 (100% Alpha) + 缩放动画 (Scale Down 0.9x)。
    *   **运行中**: 图标外圈出现旋转的光环 (Progress Ring) 或 呼吸效果。
*   **位置记忆**: 
    *   每个脚本独立记录 (x, y) 坐标。
    *   应用重启后自动飞回上次的位置。

### 2.4 录制覆盖层 (Recording Overlay)
*   **视觉**: 屏幕四周出现红色呼吸光晕 (Red Glow)，提示录制中。
*   **交互**: 
    *   拦截所有触摸事件。
    *   手指划过屏幕留下半透明轨迹线，手指抬起处留下标记点 (1, 2, 3...)。
    *   控制条显示“停止录制”与“保存”按钮。

## 3. 视觉规范 (Visual Specs)

### 3.1 配色 (Color Palette)
*   **Background**: `#121212` (深黑，OLED友好)
*   **Surface**: `#1E1E1E` (卡片背景)
*   **Primary**: `#00E5FF` (赛博青) 或 用户自定义。
*   **Secondary**: `#BB86FC` (霓虹紫)
*   **Alert/Record**: `#FF5252` (荧光红)
*   **Text**: 
    *   Primary: `#FFFFFF` (87%)
    *   Secondary: `#FFFFFF` (60%)

### 3.2 字体与图标 (Typography & Iconography)
*   **Font**: `JetBrains Mono` (用于坐标/参数，强调极客感) + `Roboto` (用于常规文本)。
*   **Icons**: Material Symbols Rounded (圆润风格，亲和力强)。

### 3.3 动画 (Motion)
*   **展开/收起**: 使用 `Spring` 弹簧物理动画，显得灵动。
*   **点击反馈**: 必须有即时的 `Ripple` 水波纹或缩放反馈。

---
此设计方案旨在利用 Android 最新的 **Jetpack Compose** 框架实现，以确保高性能的渲染和流畅的动画体验。
